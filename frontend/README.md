# GrievanceAI - Public Intelligence System

## Project Overview
India's first AI-powered grievance resolution system. Built to empower citizens and streamline governance through neural classification and real-time tracking.

## Core Features
- **Neural Classification**: AI auto-categorizes complaints with 95% accuracy.
- **Sentiment Pulse**: Detects emotional urgency for priority routing.
- **Smart Routing**: Connects cases to the right department automatically.
- **Live Transparency**: Real-time status tracking for every grievance.
- **Gamified Engagement**: Citizen rewards for civic participation.

## Technologies
- **Frontend**: React, TypeScript, Vite
- **UI & Animations**: Tailwind CSS, Framer Motion, Lucide Icons
- **Data Visuals**: Recharts
- **Components**: Shadcn/ui

## Getting Started
```sh
# Install dependencies
npm i

# Run development server
npm run dev
```
